# Change Log

Markdown2Weixin 扩展的所有重要更改都将记录在此文件中。

## [0.0.1] - 2025-11-11

### ✨ 新增功能

- 🚀 实现一键上传 Markdown 到微信公众号草稿箱
- ⚙️ 支持配置微信公众号 AppID 和 AppSecret
- 📝 自动提取文章标题（从 `# 标题` 或文件名）
- 👤 支持配置默认作者名称
- 📄 支持配置默认文章摘要
- 🎯 编辑器右键菜单集成（仅在 Markdown 文件中显示）
- 📋 上传成功后可一键复制 Media ID
- 🔔 完善的进度提示和错误反馈
- 🛡️ 配置验证和友好的错误提示

### 🎨 用户体验

- 支持未保存的 Markdown 文件上传
- 配置缺失时可直接跳转到设置页面
- 详细的上传结果展示（标题、Media ID 等）
- 中文友好的提示信息

### 🔧 技术实现

- 使用 VS Code Extension API
- 使用 Fetch API 调用微信接口
- TypeScript 类型安全
- 完善的错误处理机制
